Polling

This section contains the pre-set polling configurations. The pre-set rates are:

  • Slow: 1 Hz
  • Medium: 10 Hz
  • Fast: 20 Hz

Note: Polls must be explicitly started from the command interface.

What do you want to do?

Add Poll

Right click on the and select Add Poll.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
