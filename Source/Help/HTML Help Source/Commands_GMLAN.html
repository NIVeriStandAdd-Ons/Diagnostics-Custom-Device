Commands

The Commands section contains the command channel and related status channels for the add-on.

Command values are as follows.

  • 1 = Clear Status Bit: Clears previous command's status information.

  • 2 = Read Data Identifiers (DIDs): Reads all configured DIDs at once. Results returned in VeriStand Channels under configured DID.

  • 3 = Read Parameter Identifiers (PIDs): Reads all configured PIDs at once. Results returned in VeriStand Channels under configured PID.

  • 4 = Configure and Start PIDs (DPIDs): Sends configurations for DPIDs to the controller (UUT) and commands them to start generation.

  • 5 = Stop PIDs (DPIDs): Commands the controller (UUT) to stop generating all DPIDs.

Success: Returns the success of the command. 0 = unsuccessful and 1 = successful.

Error Code: This channel returns any error code from generating the service. If the error is from the service itself, the diagnostic error byte is return. Othewise, it is the addon error code.

Timestamp: This channel returns the absolute timestamp from when the service is finished.
