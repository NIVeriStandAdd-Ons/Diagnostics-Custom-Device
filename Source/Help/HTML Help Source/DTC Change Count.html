DTC Change Count

Enables the ECU to publish a count of how many DTCs have occurred. When an update is recieved by the addon, the Start DTCs message is automatically sent to read the updated list.

NOTE: This is a GMLAN specific service

Settings

Enable

  • 0 = Commands the ECU to stop generating the DTC change count message.
  • 1 = Commands the ECU to start generating the DTC change count message.

Status Mask

  • Bit 0 = DTC supported by calibration
  • Bit 1 = Current DTC
  • Bit 2 = test not passed since DTC cleared
  • Bit 3 = test failed since DTC cleared
  • Bit 4 = history DTC
  • Bit 5 = test not passed since current power up
  • Bit 6 = current DTC since power up
  • Bit 7 = warning indicator requested state
