DID

DIDs are data identifiers. You can read and write data identifiers. These are named Inputs and Outputs respectively.

GMLAN ONLY: DIDs, in the GMLAN context, are static in nature. An example of a DID is vehicle identification number (VIN). This is a value that doesn't change after being set. Because DIDs are static, they should be read once at the being of a test and not again.

What do you want to do?

Edit

To edit a item, right-click the object in the configuration tree and select Edit from the shortcut menu.

Delete item

To delete, right-click the object in the configuration tree and select Delete from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
