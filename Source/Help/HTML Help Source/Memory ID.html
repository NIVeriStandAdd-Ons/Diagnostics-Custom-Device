Memory Item

You can read and write memory locations by address and length. These are named Inputs and Outputs respectively.

What do you want to do?

Edit

To edit a item, right-click the object in the configuration tree and select Edit from the shortcut menu.

Delete item

To delete, right-click the object in the configuration tree and select Delete from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.

Settings

UDS Specific

Service Type

  • 1 = UDS Read Memory
  • 2 = UDS06 Read Memory

If Service Type = 1, the addiional settings are ignored. Otherwise

Address Length specifies how many bytes of the address are sent to the ECU.

Size Length specifies how many bytes of the size are sent to the ECU.

KWP2000 Specific

No additional settings.

No additional settings.
