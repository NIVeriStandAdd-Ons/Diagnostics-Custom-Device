Control DTCs

Modifies Diagnostic Trouble Code (DTC) generation behavior.

What do you want to do?

Edit

To edit a item, right-click the object in the configuration tree and select Edit from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.

Settings

UDS Specific

Type specifies the control mode:

  • 1 = On
  • 2 = Off

KWP2000 Specific

Group of DTCs specifies the group of Diagnostic Trouble Codes to be cleared.

  • 0x0000 (0) = All powertrain DTCs
  • 0x4000 (16384) = All chassis DTCs
  • 0x8000 (32768) = All body DTCs
  • 0xC000 (49152) = All network-related DTCs
  • 0xFF00 (65280) = All DTCs

GMLAN Specific

Not Supported.
