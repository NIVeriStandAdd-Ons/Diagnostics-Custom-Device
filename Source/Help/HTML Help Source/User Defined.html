User Defined

This section allows the user to create any number of user defined messages. These messages are comprised of a command/response pairing. The data of the command/response is entirely defined by the user. This allows the user to define any message(s) not already supported by add-on.

What do you want to do?

Add Message Group

To add a new item, right-click the object in the configuration tree and select Add from the shortcut menu. This group is just a container for organization.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
