Commands

The Commands section contains the command channels for the add-on. To use the command channels, first set the meta data channels to the desired value. Then set the command channel to the desired value. Command and meta data pairs are as follows.

Clear Status Bit
Clears previous command's status information (Success, Error Byte, VeriStand Error, & Data Length). It is best to call this after each command has been completed and evaluated.
Command =1
