Service Channels

This section contains the control and status channels for a service.

Trigger: This channel forces the service to be generated when the value changes from 0 to any value (ideally 1).

Success: Returns the success of the command. 0 = unsuccessful and 1 = successful.

Error Code: This channel returns any error code from generating the service. If the error is from the service itself, the diagnostic error byte is return. Othewise, it is the addon error code.

Timestamp: This channel returns the absolute timestamp from when the service is finished.
