DPIDs

DPIDs are data parameter identifiers. DPIDs are a collection of PIDs (parameter identifiers). PIDs are added together under a DPID and set to generate at one of three rates (slow, medium, or fast). DPIDs can't be more than 8 total bytes. Valid DPID ID values are typically 1..7F and 90..FE. To set up the DPIDs, first add the PIDs to the add-on. Then add the PIDs to the DPID.

What do you want to do?

Add DPID

To add a DPID, right-click the object in the configuration tree and select Add from the shortcut menu.

Import DPIDs

To import a DPID, right-click the object in the configuration tree and select Import from the shortcut menu. The import format is CSV. An example can be found at:

C:\Users\Public\Documents\National Instruments\NI VeriStand {YEAR}\Custom Devices\Diagnostics Addon\Windows\Import File Formats\DPIDs.csv

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
