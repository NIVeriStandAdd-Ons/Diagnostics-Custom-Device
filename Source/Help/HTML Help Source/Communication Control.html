Communication Control

This switchs transmission and/or reception of the normal communication messages (usually CAN messages) on or off.

Settings

All Protocols

Enable

  • 0 = Disables normal Ccmmunication.
  • 1 = Returns communication to normal.

UDS Specific

Type

  • 0 = Enable Rx and Tx
  • 1 = Enable Rx and Disable Tx
  • 2 = Disable Rx and Enable Tx
  • 3 = Disable Rx and Tx

Communication Control

  • 1 = Application
  • 2 = Network Management

KWP2000 Specific

No additional settings.

GMLAN Specific

Target Type

  • 0 = Entire Network
  • 1 = Specific ECU
