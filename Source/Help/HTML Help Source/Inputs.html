Inputs

An input is an item you read from the controller (UUT).

What do you want to do?

Add

To add a read/input item, right-click the object in the configuration tree and select Add from the shortcut menu.

Import

To import, right-click the object in the configuration tree and select Import from the shortcut menu. The import format is CSV. An examples for supported types can be found at:

C:\Users\Public\Documents\National Instruments\NI VeriStand {YEAR}\Custom Devices\Diagnostics Addon\Windows\Import File Formats

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
