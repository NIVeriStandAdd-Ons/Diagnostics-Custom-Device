Commands

The Commands section contains the command channels for the add-on. To use the command channels, first set the meta data channels to the desired value. Then set the command channel to the desired value. Command and meta data pairs are as follows.

Enable Diagnostic Session
Command = 1
MetaData1 = Mode = SubType

Disable Diagnostic Session
Forces the diagnostic session back to normal (x1)
Command = 2
No MetaData

Read Data Identifiers (DIDs)
Reads all configured DIDs at once
Command = 3
No MetaData
Results returned in VeriStand Channels under configured DID

Read Data Identifier (DID)
Command = 4
MetaData1 = ID value of the DID to read
Results returned in VeriStand Channels under configured DID

Write Data Identifier (DID)
Command = 5
MetaData1 = ID value of the DID to write
Note: Signal values pulled from configured DID

Write Memory
Command =9
MetaData1 = Address
Note: Signal values pulled from configured Memory

Read Memory
Command =10
MetaData1 = Address
Results returned in VeriStand Channels under configured Memory

Read DTCs
Command = 11
MetaData1 = Extension

  • 23 (x17) = Read Status of DTC
  • 24 (x18) = Read DTC by Status
MetaData2 = Group of DTC
MetaData3 = Mode (Not used in 23/x17)

MetaData4 = (Optional) DTC Byte Length Override

MetaData5 = (Optional) Status Byte Length Override

MetaData6 = (Optional) Failure Byte Length Override

Clear DTCs
Command =12
MetaData1 = Group of DTC

MetaData2 = (Optional) DTC Byte Length Override

MetaData3 = (Optional) Status Byte Length Override

MetaData4 = (Optional) Failure Byte Length Override

Control DTCs
Command =13
MetaData1 = Type = Group of DTC

MetaData2 = (Optional) DTC Byte Length Override

MetaData3 = (Optional) Status Byte Length Override

MetaData4 = (Optional) Failure Byte Length Override

Tester Present
Command =14
MetaData1 = Response Required

  • 0 = No
  • 1 = Yes

Disable Normal Communication
Command =15
MetaData1 = Type = Communication Control
MetaData2 = Comm Type

Return to Normal Mode
Command =16
No MetaData

Input Output Control
Command =17
MetaData1 = ID value of the IO Control to execute
MetaData2 = Mode = SubType
Note: Signal values pulled from configured IO Control
Results returned in VeriStand Channels under configured IO Control

Start Routine
Command =18
MetaData1 = ID value of the Routine to Start
Note: Signal values pulled from configured Routine
Results returned in VeriStand Channels under configured Routine

Request Routine Status
Command =19
MetaData1 = ID value of the Routine to Request Status
Note: Signal values pulled from configured Routine
Results returned in VeriStand Channels under configured Routine

Stop Routine
Command =20
MetaData1 = ID value of the Routine to Stop
Note: Signal values pulled from configured Routine
Results returned in VeriStand Channels under configured Routine

ECU Reset
Command =21
MetaData1 = Mode

Unlock ECU
Unlock the controller (UUT) with a seed key DLL.
Command =22
No MetaData

Clear Status Bit
Clears previous command's status information (Success, Error Byte, VeriStand Error, & Data Length). It is best to call this after each command has been completed and evaluated.
Command =23
No MetaData

PassThru Channel Based
Sends a raw byte array out. Allows the user to enter anything.
Command =30
MetaData1 = Service

  • Allows user to enter anything Service ID + SubType values up to 8 bytes in length.
  • e.g. Diagnostic Session would be a x10 03 where x10 is the Service and x03 is the SubType.
MetaData2 = Service Length
  • Defines the number of bytes to use in the Service ID.
  • eg. Diagnostic Session would be 2.
MetaData3 = DataLength
  • Allows user to enter any number of additional bytes to pad the raw message with. Padded value will be 0.
MetaData4 = Response Required?
  • 0 = No
  • 1 = Yes

Enable Tester Present
Starts the periodic generation of the tester present message @ 1 Hz.
Command = 31
No MetaData

Disable Tester Present
Stops the periodic generation of the tester present message.
Command = 32
No MetaData

Disconnect CAN Communication
Stops all CAN communication with the ECU and closes all references to CAN and diagnostics. This would allow a third party diagnostics tool to connect to the ECU.
Command = 33
No MetaData

Connect CAN Communication
Re-opens all references to CAN and diagnostics and starts CAN communication with the ECU.
Command = 34
No MetaData

Start Poll
Starts the pre-configuered poll.
Command = 35
MetaData1 = Poll

  • 0 = All
  • 1 = Slow
  • 2 = Medium
  • 3 = Fast

Stop Poll
Stops the pre-configuered poll.
Command = 36
MetaData1 = Poll

  • 0 = All
  • 1 = Slow
  • 2 = Medium
  • 3 = Fast
