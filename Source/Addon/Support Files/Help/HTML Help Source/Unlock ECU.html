Unlock ECU

Unlocks the ECU using various methods.

  • SeedKey DLL: Read seed -> generate key from local dll -> write key
  • MasterKey (Channel): Read seed -> generate key from VeriStand channel -> write key
  • MasterKey (Static): Read seed -> generate key from VeriStand property -> write key
  • SeedKey DLL (Windows): Read seed -> forward to Windows host -> generate key from dll -> return key to target -> write key

Note: For SeedKey DLL (Windows), Diagnostic Unlock ECU Service Launcher.vi must be manually added to the VeriStand Project as a service. The file location is:

C:\Users\Public\Documents\National Instruments\NI VeriStand \Services\Diagnostic Unlock ECU Service

To add, follow the instructions here:

https://www.ni.com/documentation/en/veristand/latest/manual/add-services/
