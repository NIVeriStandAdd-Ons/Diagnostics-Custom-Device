Diag Session Control

Sets up the ECU in a specific diagnostic mode.

Settings

All Protocols

Enable

  • 0 = Stops the diagnostic session.
  • 1 = Starts the diagnostic session.

UDS

Mode indicates the diagnostic mode into which the ECU is brought. Ignored when disabling the session. The values are application specific. The usual values are:

  • x1 = defaultSession
  • x2 = ECUProgrammingSession
  • x3 = ECUExtendedDiagnosticSession

KWP2000

Mode indicates the diagnostic mode into which the ECU is brought. Ignored when disabling the session. The values are application specific.
