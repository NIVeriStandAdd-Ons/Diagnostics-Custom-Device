Poll

This section allows supported services to be scheduled at pre-set rates. Supported services are:

  • Read Data Identifiers
  • Read Parameter Identifiers (GMLAN Only)

The pre-set rates are:

  • Slow: 1 Hz
  • Medium: 10 Hz
  • Fast: 20 Hz

Note: Specific IDs can only be used in one poll. As an example, if the data identifier (DID) xA0 is already selected in the Fast poll, it will not be available to select in Medium or Slow.

Note: Polls must be explicitly started from the command interface.

What do you want to do?

Edit Poll

Right click on the desired poll rate and select Edit.

Delete item

To delete, right-click the object in the configuration tree and select Delete from the shortcut menu.

Settings

All Protocols

Enable

  • 0 = Disables periodic generation.
  • 1 = Enables periodic generation.

Note: Set the Trigger channel to 1 to apply the setting value.
