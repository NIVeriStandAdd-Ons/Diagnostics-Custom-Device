ECU Reset

Resets the ECU.

Settings

UDS Specific

Mode indicates the reset mode:

  • x1 = Hard Reset
  • x2 = Key Off On Reset
  • x3 = Soft Reset
  • x4 = Enable Rapid Power Shut Down
  • x5 = Disable Rapid Power Shut Down

KWP2000 Specific

Mode indicates the reset mode:

  • x1 = PowerOn - Simulates reset that most ECUs perform after the ignition OFF/ON cycle.
  • x2= Power On While Maintaining Communication - Simulates reset that most ECUs perform after the ignition OFF/ON cycle but maintains communication with the client (tester).
  • 03-7F = Reserved
  • 80-FF = ManufacturerSpecific

GMLAN Specific

Not supported.
