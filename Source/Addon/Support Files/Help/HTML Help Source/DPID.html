DPID

DPIDs are data parameter identifiers. DPIDs are a collection of PIDs (parameter identifiers). PIDs are added together under a DPID and set to generate at one of three rates (slow, medium, or fast). DPIDs can't be more than 8 total bytes. Valid DPID ID values are typically 1..7F and 90..FE. To set up the DPIDs, first add the PIDs to the add-on. Then add the PIDs to the DPID.

NOTE: This is a GMLAN specific service

What do you want to do?

Edit DPID

To edit a DPID, right-click the object in the configuration tree and select Edit DPID from the shortcut menu.

Delete item

To delete, right-click the object in the configuration tree and select Delete from the shortcut menu.

Settings

Enable

  • 0 = Stops the DPID configuration.
  • 1 = Configures the DPID with the ECU and commands the ECU to start generation.
