Routines

Routines are functions (programs) that run on the controller (UUT). You can start and stop them as well as request results. They are referenced by ID.

What do you want to do?

Add

To add a new item, right-click the object in the configuration tree and select Add from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
